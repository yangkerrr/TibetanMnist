#TibetanMnist(手写藏文数字识别)
Realize handwritten Tibetan number recognition based on LeNet, and use pyqt to realize an interface system.
基于LeNet实现手写藏文数字识别，并使用pyqt实现一个界面系统。

##Requirements
Paddle 2.0
PyQt5
